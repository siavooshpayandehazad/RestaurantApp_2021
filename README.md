# RestaurantApp
A small app for an imaginary restaurant...

In order to run the app you need to install virtualenv on your machine. Then you can run it like this:

```
cd RestaurantApp
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 main.py
```


# Your Task
